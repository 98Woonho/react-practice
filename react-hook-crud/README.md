# React Hook을 이용한 CRUD

## React Hook이란?
React Hook은 Reack Class Component에서 이용하던 코드를 작성할 필요 없이, 함수형 컴포넌트에서 다양한 기능을 사용할 수 있게 만들어준 라이브러리이다. Hook에는 종류가 여러가지가 있는데, 그 중에서 useState를 사용 할 것이다.
